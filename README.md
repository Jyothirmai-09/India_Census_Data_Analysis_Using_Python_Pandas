# India_Census_Data_Analysis_Using_Python_Pandas

## 📌 Overview

This project analyzes the India Census 2011 dataset using **Python** and the **Pandas** library to explore demographic, population, literacy, workforce, and religion-based statistics across different states and districts of India. Through data cleaning, filtering, grouping, and exploratory data analysis (EDA), the project demonstrates how Python can be used to extract meaningful insights from large-scale census data.

---

## 📂 Dataset Information

The dataset contains district-wise and state-wise census records collected during the **India Census 2011** survey.

### Key Features

* State Name
* District Name
* Population
* Male & Female Population
* Literate Population
* Workers & Employment Statistics
* Male Workers & Female Workers
* Religious Population Distribution
* Educational Statistics
* Age Group Distribution

Each record represents census information for a specific district in India.
